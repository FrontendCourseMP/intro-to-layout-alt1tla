Cоздаёт псевдоэлемент, который является последним потомком выбранного элемента.
Before
Является первым потомком выбранного элемента. Часто используется для добавления косметического содержимого в элемент с помощью свойства content.
Active
Псевдокласс :active соответствует элементу в момент, когда он активируется пользователем.
Focus
CSS псевдокласс :focus применяется, когда элемент (такой как input формы) получает фокус.
Left
Псевдокласс CSS :left используется с at-правилом @page, предоставляет все левые страницы печатного документа.
